macOS

Windows Installation

Option 1: Using the Installer (Recommended)

  1. Download the installer (.exe file)
  2. Right-click and "Run as administrator"
  3. Follow the installation wizard
  4. GPULang will be added to your PATH automatically

Option 2: Portable ZIP

  1. Download and extract the ZIP file
  2. Add the bin/ directory to your PATH environment variable

Verify Installation:

gpulangc --version

macOS Installation

Step 1: Choose Your Architecture

  • Apple Silicon (M1/M2/M3): Download the ARM64 version for better performance
  • Intel Macs: Download the x64 version for Intel-based Macs
  • Unsure? Go to Apple Menu → "About This Mac" to check your processor

Option 1: Using the DMG (Recommended)

  1. Download the appropriate .dmg file for your Mac architecture
  2. Double-click to mount the disk image
  3. Drag GPULang to your Applications folder
  4. Add /Applications/GPULang.app/Contents/MacOS to your PATH

Option 2: TAR.GZ Archive

tar -xzf gpulang-v1.0.0-macos-[architecture].tar.gz
sudo mv gpulang /usr/local/bin/

Architecture Detection (Optional):

uname -m

Returns "arm64" for Apple Silicon or "x86_64" for Intel

Verify Installation:

gpulangc --version

Linux Installation

Option 1: AppImage (Recommended)

  1. Download the .AppImage file
  2. Make it executable and run:
chmod +x gpulang-v1.0.0-linux-x64.AppImage
./gpulang-v1.0.0-linux-x64.AppImage --install

Option 2: TAR.GZ Archive

tar -xzf gpulang-v1.0.0-linux-x64.tar.gz
sudo cp gpulang/bin/* /usr/local/bin/

Verify Installation:

gpulangc --version
